## 📝 Descrição
O **WPS Solutions** é um website institucional moderno e responsivo focado na apresentação de serviços de infraestrutura tecnológica. O projeto utiliza uma identidade visual baseada no "Royal Blue" para transmitir confiança e autoridade em soluções de engenharia e conectividade.

## 🚀 Funcionalidades
* **Design Responsivo:** Interface adaptável para dispositivos móveis e desktops via Tailwind CSS.
* **Navegação Inteligente:** Menu fixo com rolagem suave entre as seções de Início, Sobre, Serviços e Contato.
* **Carrosséis Dinâmicos:** Exibição interativa de portfólio de clientes e catálogo de serviços utilizando Owl Carousel 2.
* **Seções Institucionais:** Áreas dedicadas à história da empresa, diferenciais competitivos e prova social.
* **Formulário de Contato:** Interface pronta para captação de leads e solicitações de orçamento.

## 🛠️ Tecnologias Utilizadas
* **Frontend:** HTML5 semântico.
* **Estilização:** [Tailwind CSS](https://tailwindcss.com/) (Customizado via JS).
* **Componentes Visuais:** [Owl Carousel 2](https://owlcarousel2.github.io/OwlCarousel2/) para sliders responsivos.
* **Ícones:** [Bootstrap Icons](https://icons.getbootstrap.com/).
* **Scripts:** [jQuery](https://jquery.com/) para manipulação de eventos e inicialização de plugins.
